Passphrase → PBKDF2 (SHA-256) → AES-256-GCM (TOKEN v3)

Format token: v3.<iters>.<salt>.<nonce>.<ciphertext>.<tag> (Base64URL, bỏ "=")
“Khó biết” phụ thuộc chủ yếu vào PASS mạnh + iterations cao. Token public vẫn OK.
Gợi ý: PASS nên 20–32 ký tự. Iterations 600k là mức mạnh nhưng có thể hơi chậm trên máy yếu.
